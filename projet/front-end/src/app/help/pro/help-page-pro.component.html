Pour créer des quiz, appuyez sur le bouton "Ajouter un quiz" et remplissez le formulaire.

Pour voir vos quiz créés, appuyez sur le bouton "Mes quiz".

Pour retourner sur la page d'accueil, appuyer sur l'icône du personnage.

Pour voir les suivis de vos patients, appuyez sur le bouton "Mes suivis".

Pour ajouter des patients dont vous voulez voir le suivi, appuyez sur le bouton "Mes suivis" puis sur "Ajouter un patient" et donnez le nom du patient à ajouter.
