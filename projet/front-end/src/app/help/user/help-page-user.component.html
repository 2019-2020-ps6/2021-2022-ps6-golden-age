Pour faire des quiz, appuyez sur le bouton "Sélectionner un quiz" puis choisissez le quiz qui vous plaît.

Pour voir les résultats de vos quiz, appuyez sur le bouton "Mes statistiques".

Pour retourner sur la page d'accueil, appuyer sur le bouton "Accueil" ou sur l'icône du personnage en haut à gauche de la page.

Pour modifier vos configurations, cliquez sur l'engrenage en haut de votre écran. Vous pourrez modifier la taille des caractères mais aussi choisir la voix que vous préférez entendre pour lire les quiz.
